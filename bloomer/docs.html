Bloomer is a layout built on the Bulma framework, which is definitely what you might think it's named after.

You should know basic HTML/CSS to use this layout, and if you want to know how to customize it, great! That's what this page is for. Any further (advanced) customization requires reading Bulma's comprehensive docs here: https://bulma.io/documentation

Install

You can install the basic layout here.

Repository

You can also download the layout from the repository on Github:
https://github.com/nocturnenebula/cf-layouts/tree/main/bloomer/

I highly recommend you keep a local copy as a backup and create a copy of the layout before modifying it.

Features

  • Easy to install and customize
  • Thoroughly documented and annotated
  • Dropdown for extra outside links
  • Hassle free mobile readiness
  • Squeaky clean modern design

Layout CSS

All of the CSS is located in the Layout CSS section. It uses CSS variables so that all of your customization stays in one place and you don't have to manually change every line of CSS that shares the same properties.

These are the default variables you can edit.

      :root{ /* learn more about it here: https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties */

      --body-bg-color:#FFF; /*this changes the overall page's background color*/
      --body-bg-image: url(''); /*add the body image here*/
      --body-bg-position: center repeat; /*add the body image positioning here*/

      --body-font: BlinkMacSystemFont,-apple-system,"Segoe UI",Roboto,Oxygen,Ubuntu,Cantarell,"Fira Sans","Droid     Sans","Helvetica Neue",Helvetica,Arial,sans-serif ; /*change the font-family of the entire site here*/
      --body-font-size: 16px; /*change the font size of the entire site here*/
      --body-color:#4a4a4a; /*change the font color of the entire site here*/

      --link-color:#3273dc; /*change the links color here*/
      --link-hover-color: #363636  /*change the links hover color here*/;

      --banner-image:  url('http://placehold.it/1600x350?text=Banner+Image'); /* replace the background here */
      --banner-position: center no-repeat;
      --banner-bg-color: #FFF; /*-- you can change the color of the banner bg here (if using transparent images) */
      --banner-size: cover; /* this stretches the banner out to cover the whole area */
      --banner-height: 350px; /*-- you can change the height of the banner here */
  }
 

I've labelled them as thoroughly as possible, but if you still need further assistance, check the forums.

Navigation

The menu controls the entire navigation, and it appears on every page. By default, the navigation displays the logo, links and then a dropdown featuring your own custom links.

Logo

You can change the logo or remove it all together by deleting this code:
    <a class="navbar-item" href="/"> <!-- replace the image with your own comic logo -->
      <img src="https://nocturnenebula.github.io/sj-layouts/bloomer/logo.png" width="128" height="28">
    </a>
 

Links

Links use the menu loops and switches by default, but let's say you want to shorten "Latest Page" to just "Latest" or you don't want to display every custom page. Remove this section:

  <!-- BEGIN links -->
    <a class="navbar-item" href="{links.URL}">{links.TEXT}</a>
  <!-- END links -->

And replace it with:

     <a class="navbar-item" href="/">Home</a>
    <a class="navbar-item" href="{FIRST_COMIC_URL}">First</a>
    <a class="navbar-item" href="{LATEST_COMIC_URL}">Latest</a>
    <a class="navbar-item" href="/archive">Archive</a>
    <a class="navbar-item" href="#">Custom Link 1</a>
    <a class="navbar-item" href="#">Custom Link 2</a>
    <a class="navbar-item" href="#">Custom Link 3</a>

Dropdown

If you don't want to use the dropdown, then remove this entire section.
  <!-- change the link names and add the URLs for the dropdown here -->

           <div class="navbar-item has-dropdown is-hoverable">
             <a class="navbar-link">
               Dropdown <!-- change the name to something like more, further reading, etc. -->
             </a>

             <div class="navbar-dropdown"> <!-- custom outgoing links go here -->
               <a class="navbar-item" href="#your link here">Link </a>
               <a class="navbar-item" href="#your link here">Link </a>
               <a class="navbar-item" href="#your link here">Link </a>

               <hr class="navbar-divider">

               <a class="navbar-item" href="#your link here">Separated Link </a>
               <a class="navbar-item" href="#your link here"><span class="icon has-text-danger"><i class="fas fa-heart"></i></span><span>Icon Link</span></a>


             </div>
            </div>
 

Icons

Icons are located in <div class="navbar-end">.

You can add, edit or delete them by removing <p class="control"> --- HTML, etc. --- </p>.
