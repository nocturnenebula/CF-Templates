Bloomer is a layout built on the Bulma framework, which is definitely what you might think it's named after.

You should know basic HTML/CSS to use this layout, and if you want to know how to customize it, great! That's what this page is for. Any further (advanced) customization requires reading Bulma's comprehensive docs here: https://bulma.io/documentation

Install

You can install the basic layout here.

Repository

You can also download the layout from the repository on Github:
https://github.com/nocturnenebula/cf-layouts/tree/main/bloomer/code

I highly recommend you keep a local copy as a backup and create a copy of the layout before modifying it.

Features

  • Easy to install and customize
  • Fancy features for the Overview page
  • Dropdown for extra outside links
  • Really cool font icons
  • Hassle free mobile readiness
  • Squeaky clean modern design

Overall

Overall is self-explanatory. This will control what elements appear on every single page of your webcomic.

Header/Banner

By default, there is a header or "banner" for the layout. You can remove it by removing this section:

<section class="banner hero is-medium">
  <div class="hero-body">
    <div class="container">
      <p class="title">
      </p>
    </div>
  </div>
</section>

Menu

The menu controls the entire navigation, and it appears on every page. By default, the navigation displays the logo, links and then a dropdown featuring your own custom links.

You can also change the color of the navigation bar (but requires extra CSS coding), learn it here.

Logo

You can change the logo or remove it all together by deleting this code:

<a class="navbar-item" href="/">
	<img src="http://bloomer-layout.cfw.me/files/logo.png" width="128" height="28">
</a>

Links

By default, the links for the navigation are Home, First (comic), Latest (comic), Archive, Blog, and then your extra web pages. There is also a dropdown for outgoing links and for the scroll view, which is located in the next section. You can edit, delete or add more links by editing this section:

<div id="navbar-menu" class="navbar-menu">
      <div class="navbar-start">

        <a class="navbar-item" href="/">Home</a>
        <a class="navbar-item" href="/comics/1">First</a>
        <a class="navbar-item" href="/comics/">Latest</a>
        <a class="navbar-item" href="/archive/">Archive</a>
        <a class="navbar-item" href="/blog/">Blog</a>

        [l:extrapages]
          <a class="navbar-item" href="[v:l.link]">[v:l.title]</a>
        [/]
      </div>
		</div>

Dropdown

The dropdown helps save you space from all those extra pages, and to provide outgoing links. If you don't want to use the dropdown, then remove this entire section.

<div class="navbar-item has-dropdown is-hoverable">
		<a class="navbar-link">Links</a>
		<div class="navbar-dropdown">
			<a class="navbar-item">Link</a>
			<a class="navbar-item">Link</a>
			<a class="navbar-item">Link</a>
			<hr class="navbar-divider">
			[c:infinitescrollenabled]
				<a class="navbar-item" href="[v:infinitescrolllink]">Scroll View</a>
			[/]
		</div>
	</div>

To keep the Infinite Scroll View, copy and paste this into navbar-start: 

[c:infinitescrollenabled]
				<a class="navbar-item" href="[v:infinitescrolllink]">Scroll View</a>
			[/]
		</div>

Icons

All of the icons are hosted through Fontawesome. Icons are located in <div class="navbar-end">, but they can be added anywhere onto your layout.

You can edit or copy the icons by changing any of the sections that look like this:

<p class="control">
	<a href="#" target="_blank">
		<span class="icon"><i class="fab fa-patreon"><i></span>
	</a>
</p>

You can also delete the ones you don't want! Just remove the entire p block.

To change the icon, look for any section that looks like <i class="fab fa-brandname"></i>. Change the link, and then you can start looking for the right brand icon here: https://fontawesome.com/icons?d=gallery&s=brands

Sometimes they won't have the brand you're looking for, so you'll have to compensate. Like for example, you can use a paw for FurAffinity.

Content

There is an itty bitty code for the content that looks like this:

[c:isextrapage]
	<h2 class="title is-2">[v:pagetitle]</h2>
	[/]
<!--layout:[content]-->

<h2 class="title is-2">[v:pagetitle]</h2> is what adds a title to every extra page you make. If you don't want to use it then just remove that entire line of code.

Footer

The footer is very simple and it's at the bottom of every page. To edit the copyright section, go to Manage > Edit Webcomic Profile/Info > Copyrights.

Layout CSS

All of the CSS is located in the Layout CSS section. It uses CSS variables so that all of your customization stays in one place and you don't have to manually change every line of CSS that shares the same properties.

These are the default variables you can edit:

  :root{ /* learn more about it here: https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties */

--body-bg-color:#FFF; /*this changes the overall page's background color*/
--body-bg-image: url(''); /*add the body image here*/
--body-bg-position: center; /*add the body image positioning here. this can be two values (ex: top center) */
--body-bg-repeat: repeat; /* add the body image repeatability here. you can change it to repeat, repeat-x, repeat-y, etc. */
--body-bg-size: auto; /*this changes how you want the bg image to stretch out. */
--body-font: BlinkMacSystemFont,-apple-system,"Segoe UI",Roboto,Oxygen,Ubuntu,Cantarell,"Fira Sans","Droid Sans","Helvetica Neue",Helvetica,Arial,sans-serif ; /*change the font-family of the entire site here*/
--body-font-size: 16px; /*change the font size of the entire site here*/
--body-color: #4a4a4a; /*change the font color of the entire site here*/

--link:#3273dc; /*change the links color here*/
--link-hover: #363636;  /*change the links hover color here*/

--banner-bg-color: #00d1b2; /*-- you can change the color of the banner bg here (if using transparent images) */
--banner-image:  url('http://placehold.it/1600x350/00d1b2/FFFFFF/?text=Banner+Image'); /* replace the background here */
--banner-position: center; /*-- by default, the banner is centered. this is where you can change it --*/
--banner-repeat: no-repeat; /*-- by default, it's set not to repeat. you can change it to repeat, repeat-x, repeat-y, etc. --*/
--banner-size: cover; /* this stretches the banner out to cover the whole area */
--banner-height: 350px; /*-- you can change the height of the banner here */

--comic-nav: #4a4a4a; /*This changes the color of the comic navigation! */
--comic-nav-hover: #777; /*This is what it changes to when you hover over it! */

--footer-bg-color: #fafafa; /* this changes the footer's background color */
--footer-bg-image: url(''); /* changes the footer background image */
--footer-bg-position: top center; /* changes the positioning of the background image */
--footer-bg-repeat: repeat;
--footer-bg-size: contain;
--footer-link: #3273dc;
--footer-link-hover: #363636;
  }

I've labelled them as thoroughly as possible, but if you still need further assistance, check the forums.

Comic Page

The Comic Page is aptly named, it's where all of your code for the comic pages go.

Comic Navigation

By default, the navigation uses FontAwesome icons which scale with the webpage. If you want to remove the text so it's just the arrows, then remove all <span> classes including the text inside of them. If you want to use your own custom images, then replace the <i> tags with your img tags.

Comic Comments

The comments section consists of the author's comment, the comments and replies which nest up to 5 comments but shrink down on mobile.

Archive

Archive is where all of your comics are listed in a table. There's not much to customize in this section, but you're welcome to edit it.

Overview

Overview is an optional page for the front page of your comic. By default, it's not activated and so you'll need to activate it by going to your webcomic's Manage > Edit Settings > Front page.

Introduction

The introduction of the overview page consists of the comic's title, the slogan, the description, two buttons for the first and latest comic, and finally the image of the latest comic.

By default, the title, slogan and description is what you added in Manage > Edit Webcomic Profile/Info.

To change the button colors, see this for reference: https://bulma.io/documentation/elements/button/#colors

Cards

Cards are a little feature added to make the overview page more interesting. Use them for listing links, cast members, etc. or don't use them at all! Just delete the entire code beginning with <div class="hero is-light">.

Search

Search is the landing page for any search results. You can change the error message in search (or make it prettier) by changing this section:

<p>Oops, looks like we didn't find anything. Try again!</p>

Blog Archive

Blog Archive controls the appearance of the list of blog posts. By default, the blog post's contents aren't visible, but if you want the archive to include the blog post's contents then add this section after <strong><a href="[v:l.profilelink]" class="author">[v:l.authorname]</a></strong> <small>[v:l.posttime]</small>:

<div class="content mt-5">
      [v:blog]
    </div>

Blog Display

Blog Display controls the blog post's appearance. By default, it consists of the blog post itself with navigation buttons on its right, and then the comments.

Blog Comments

The comments section for the blog is the same as the comments section for the comic comments. See Comic Comments.

Error Page

Self-explanatory. This is what will show up when someone reaches a page that doesn't exist (or some other error-related reason). You can add images and other content to this section.
